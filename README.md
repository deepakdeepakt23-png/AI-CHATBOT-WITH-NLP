PROJECT OVERVIEW :

An AI chatbot is a software program designed to simulate human conversation and answer user queries automatically. Chatbots are widely used in customer service, education, and online assistance systems. This project focuses on building a simple AI chatbot using Natural Language Processing (NLP) techniques in Python.

OBJECTIVE :

The main objective of this project is to develop a chatbot that can understand user queries and respond with appropriate answers. The chatbot uses the NLTK (Natural Language Toolkit) library for basic Natural Language Processing tasks such as tokenization and text processing.

TOOLS AND TECHNOLOGIES USED :

* Python – Programming language used to develop the chatbot.

* NLTK (Natural Language Toolkit) – Used for Natural Language Processing.

* VS Code – Code editor used for writing and executing the Python script.

WORKING :

The chatbot works by receiving a user query through the command line interface. The user input is converted into lowercase text and processed using the NLTK tokenizer. The program then compares the user input with predefined question-answer pairs stored in a dictionary.

IMPLEMENTATION :

The chatbot is implemented using a Python script. A dictionary called qa_pairs stores multiple predefined questions and answers related to computer science and technology topics. The chatbot continuously waits for user input using a loop and checks whether the entered question exists in the dictionary.
